Prerequisite: in order to run this application you will need to install the LibraryManager2 API. This is the React front-end of the application.

Getting Started

1. Clone the repo
  git clone https://github.com/tam4385/LibraryManager2.git

2. install dependencies
  npm i

3. Run start
  npm start

Tech Stack:

  Reack JS

  Redux

  Node JS/ Express

  Mongoose

The React front end for a library manager app. Keep track of your books with this simple app.
